About this site

Welcome to this blog site. This site is simple example of blog site, which includes a few posts. Each post has its own detail, where is the content of the post.
In the navigation bar, you can search posts by name. Each post has also one category and can have one or more tags.
If you are superuser or staff user you can add, delete or edit posts, categories and tags. You can see this options in admin dashboard or if you are logged in, you have these options directly on the site.

You can login by click here.

Please remember you cannot create new user on this site. If you want to create user you need to contact admin or go to my MultiApp platform, where is the same blog site but everyone can create new account.
